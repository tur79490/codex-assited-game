# Lantern Logic Quest

Lantern Logic Quest is a single-player browser adventure game where the player guides Mira through 10 illustrated stages, answers harder general knowledge questions, uses hints when needed, and reaches a treasure vault at the end.

## Features

- Home screen with a start button
- 10 logic and knowledge-based stages
- Richer visual environments with both outdoor and indoor scenes
- Haunted castle rooms with cobwebs, candles, and moonlit details
- Mouse and trackpad-only gameplay
- Hint button on every stage
- Restart button during play
- Confetti celebration after winning
- Interactive treasure chest that gives 1 gold coin per click
- Greed reset rule if the player takes more than 10 coins

## How to Run the App

1. Open the project folder on your computer.
2. Locate the `index.html` file.
3. Double-click `index.html`.
4. If your computer asks which browser to use, choose Google Chrome.
5. On the home screen, click `Start Quest` to begin the game.

## Requirements

- Google Chrome browser
- A mouse or trackpad
- No internet connection required after the files are available locally
- No server, package manager, or external framework required
